Math Equation Generator
This is a program that I created that generates simple and random math equations. If you get the question wrong, the program will tell you the correct answer and ask if you want to play again. If you get the question correct, the program will tell you it is the correct answer and ask if you want to play again. Insult Generator
This insult generator was created for a school project. It gets the user to input their name or a random name in general, pick how many sentences of insults they would like generated, and how many insults they want to use per sentence. The program then takes the users input and creates an offensive list of insults based on the users input.
